# Fondos de Limine

Esta carpeta contiene los fondos utilizados por el gestor de arranque **Limine**.\
El archivo activo usado por el tema se define en los scripts o configuraciones correspondientes.

## Archivo activo

- **boot-bg.jpg**\
  Fondo actual utilizado durante el arranque del sistema.

## Referencias

- **Fuente:** [Pixiv – Artwork #134239802](https://www.pixiv.net/en/artworks/134239802)

## Notas

- Los nombres de archivo pueden cambiar libremente sin afectar los scripts, siempre que `boot-bg.jpg` exista.
- Las imágenes no se incluyen en el control de versiones para evitar peso innecesario y respetar derechos de autor.
- Este archivo (`README.md`) mantiene la trazabilidad de las fuentes originales.
